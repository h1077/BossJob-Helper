# 小臣版 AI-Boss 海投助手 V2.0.0.0

找工作真的很累，每天在 BOSS 直聘上刷职位、投简历、等回复，一遍又一遍地重复这些操作，时间长了真的会让人崩溃。我重构这个工具的初衷很简单：**让机械化的工作交给脚本去做，把更多的时间和精力留给准备面试、提升自己这些更重要的事情。**

这个工具不是什么黑科技，就是一个帮你自动点击、智能筛选、AI 辅助回复的浏览器脚本。它能帮你做的事情，你自己也能做，只是它会做得更快、更准确，而且永远不知疲倦。

我自己在找工作时，每天花几个小时手动投递数百份简历，却因为投给了大量一年前发布的僵尸岗位，回复率不到 10%。后来我基于开源项目重构了这个 V2.0 脚本，加入了**多平台大模型接入、本地简历智能提取、多城市自动轮询**等功能。只需初次配置，剩下的交给 AI 自动处理，回复率得到了肉眼可见的提升。
**所以，这个工具不是要取代你，而是要武装你，让你把时间花在更有价值的面试准备上。**

## 🚀 核心功能：它能帮你做什么？

### 1. 本地简历上传与 AI 智能解析（V2.0 独家）
不再需要手动编写长篇大论的自我介绍！脚本内置了纯前端的文档解析引擎：
* 支持直接上传 **PDF、Word、TXT** 格式的简历文件。
* 结合你配置的 AI 大模型，一键自动提炼核心技能、工作经验与个人优势。
* **自动生成 4 条个性化打招呼语**，让你的首次沟通专业且吸睛。

### 2. 灵活的多模型 AI 智能回复
当 HR 回复你的消息时，AI 会根据 HR 的问题，**结合你刚上传的简历信息**生成自然、专业的回复内容。
* 支持自定义接入：**硅基流动、DeepSeek、火山引擎、OpenAI** 等主流大模型 API。
* 提供一键预设功能，零门槛快速上手。
* AI 角色提示词完全可自定义，轻松应对各种面试场景。
<img width="956" height="1426" alt="image" src="https://github.com/user-attachments/assets/2808c372-4e58-480e-af5c-09345cef38aa" />

### 3. 自动批量投递与多城市轮询
* 设定好职位关键词、工作地点、薪资范围，脚本自动扫描页面并逐个点击“立即沟通”。
* **支持多城市自动无缝切换**，当前城市投递完毕后，自动跳转至下一个目标城市，彻底解放双手。

### 4. 精准过滤与防重复机制
* **HR 活跃度过滤**：精准识别 HR 的在线状态（支持筛选“在线”、“刚刚活跃”等），直接过滤掉长期不在线的僵尸岗位。
* **猎头过滤**：一键开启排除猎头岗位，直面企业 HR。
* **防重复投递**：本地大容量缓存记录已投递的岗位与 HR，下次运行自动跳过，绝不浪费每天宝贵的 150 次投递配额。

### 5. 自动发送附件/图片简历
HR 查看附件简历往往需要消耗平台配额，很多人因此错失机会。
* 脚本支持**配置多张 JPG 图片简历**。
* 首次沟通时自动打包发送图片简历，大幅降低 HR 的查看门槛，提高沟通意愿。

## 🛠 技术架构与技术栈

本脚本采用单文件模块化设计，无外部服务器依赖（除 AI 接口外），所有核心逻辑均在本地浏览器执行，最大程度保证你的隐私安全。
<img width="662" height="980" alt="image" src="https://github.com/user-attachments/assets/4533e4c5-b6ee-414e-a5d8-d657df38579a" />

**前端架构：**
* **核心业务类**：`Core`（主流程控制）、`HRInteractionManager`（沟通与简历发送逻辑）。
* **UI 组件**：原生 DOM 构建的轻量级可拖拽悬浮面板、暗色/亮色主题系统。
* **数据存储**：`StorageManager` 与自动截断降级存储方案，完美解决长文本 `localStorage` 容量溢出问题。
* **文件解析**：原生 `FileReader` 配合多重降级正则匹配，实现无后端的 PDF/Word 文本提取。

**依赖与要求：**
* **运行环境**：Tampermonkey / ScriptCat 脚本管理器。
* **浏览器支持**：Chrome, Edge, Firefox, 360 极速浏览器 X（建议最新版）。
* **加密库**：引入 `crypto-js` 保障本地数据处理安全。

## 📦 怎么安装？

### 第一步：安装浏览器脚本管理器
推荐使用 **ScriptCat（脚本猫）** 或 **Tampermonkey（油猴）**。
安装后，请务必在浏览器的扩展管理页面打开以下权限：
* 允许访问文件 URL
* 允许在隐身模式下运行（按需）

### 第二步：安装脚本
* **一键安装**：访问 [GreasyFork 或项目的 GitHub 主页](https://github.com/DYxiaochen/AI-BossJob)，点击安装按钮。
* **手动安装**：下载 `Boss_helper.js` 的源码，打开脚本管理器的“添加新脚本”，粘贴代码并按 `Ctrl+S` 保存。

## 💡 怎么使用？

1. **登录 BOSS 直聘**：确保账号已登录。
2. **配置 AI 与简历**：
   * 打开职位列表页（`/web/geek/jobs`），点击右侧悬浮面板的 **“AI 配置”**。
   * 填入你的 DeepSeek 或其他平台的 API Key。
   * 在“插件设置”中上传你的 PDF/Word 简历，点击 **“AI 分析简历”**，脚本会自动为你生成打招呼语并保存分析结果。
3. **开启系统招呼语**：在 BOSS 的设置页面，确保“打招呼”功能开启。
4. **设置图片简历（强烈推荐）**：在高级设置中上传处理过联系方式的 JPG 简历图片，并开启自动发送。
5. **设置筛选条件并启动**：在面板中输入职位关键词、目标城市，选择好 HR 活跃时间（如“在线”），点击 **“启动海投”**，接下来喝杯咖啡，静候佳音。

## ⚠️ 使用建议与防风控指南

* **控制投递频率**：BOSS 直聘每日打招呼上限为 150 次。建议分时段投递（如早 9 点、下午 2 点、晚 5 点各投 50 个），**切勿一次性疯狂刷满**，以免触发系统风控导致账号限流。
* **重视 HR 在线时间**：强烈建议将过滤条件设为“在线”或“刚刚活跃”，投递给半个月未上线的 HR 纯属浪费配额。
* **图片简历务必脱敏**：上传的图片简历中**必须删除或打码处理手机号、微信号、邮箱等联系方式**！否则极易触发平台的防引流检测，导致投递失败甚至封号。
