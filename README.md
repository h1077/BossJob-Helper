# BossJob-Helper — BOSS 直聘智能求职助手

> AI 定制招呼语 · 模拟面试 · 投递追踪看板 · 简历评分优化

找工作最累的不是面试，是每天在 BOSS 直聘上重复点击"立即沟通"、复制粘贴千篇一律的招呼语、投完就忘不知道哪家回复了。

**BossJob-Helper** 把机械化操作交给脚本，把数据管理交给桌面端——你只管准备面试。

---

## 🏗️ 架构：双端协作

```
┌─ 浏览器（Tampermonkey 脚本）─────┐      ┌─ 桌面端（Flask + SQLite）──────┐
│                                  │      │                               │
│  · 自动滚动职位列表               │      │  · AI API 代理（隐藏 Key）     │
│  · 点击"立即沟通"                 │ ───→ │  · 简历六维度评分              │
│  · 发送 JD 定制招呼语             │ ←─── │  · JD 招呼语生成 + 缓存         │
│  · 发送附件/图片简历              │      │  · AI 模拟面试                 │
│  · AI 自动回复 HR 消息            │      │  · 投递数据看板                │
│  · 投递数据实时回传               │ ───→ │  · CSV/JSON/Excel 导出         │
│                                  │      │  · SQLite 持久化存储           │
└──────────────────────────────────┘      └───────────────────────────────┘
```

| 职责 | JS 脚本 | 桌面端 |
|------|:--:|:--:|
| 页面操作（点按钮、发消息、填表单） | ✅ | ❌ |
| AI API 代理（隐藏 Key） | ❌ | ✅ |
| 简历评分 + AI 优化 | ❌ | ✅ |
| JD 定制招呼语生成 | 降级可用 | ✅ 主力 + 缓存 |
| AI 模拟面试 | ❌ | ✅ |
| 投递看板 + 数据导出 | ❌ | ✅ |
| SQLite 持久化 | ❌ | ✅ |
| 实时聊天监听 + AI 回复 | ✅ | ❌ |
| 图片简历发送 | ✅ | ❌ |

**脚本不可用时**：桌面端仪表盘仍可查看历史数据、编辑简历、模拟面试。
**桌面端不可用时**：脚本自动降级为直连 AI API + localStorage 存储，不影响投递。

## ✨ 功能详解

### 1. JD 千岗千面定制招呼语

每个岗位单独生成专属招呼语，不是复制粘贴的固定文案。

```
普通招呼语："您好，我对这个岗位感兴趣，希望能进一步沟通"

JD 定制招呼语（AI 生成）：
"熟悉 React、TypeScript，做过大型 SPA 架构设计，
主导过日均 100 万 PV 的管理后台重构，与贵岗位要求高度匹配"
```

**三级智能降级**：

| 条件 | 模式 | 缓存 |
|------|------|:--:|
| 有简历 + 有 JD | 全量 Prompt（匹配技能、项目经验） | ✅ |
| 无简历 + 有 JD | 降级 Prompt（基于 JD 表兴趣） | ✅ |
| 无 JD | 使用预设固定招呼语 | — |

同一岗位只生成一次，后续投递直接从 SQLite 读缓存，不浪费 AI Token。

### 2. AI 模拟面试

AI 扮演 HR，根据**你选的岗位 JD + 你的简历**，分 6 个阶段进行模拟面试：

```
开场 → 技术深挖 → 项目经验 → 综合素质 → 候选人反问 → 结束
```

面试结束后自动输出评估报告：
- **综合评分**（0-100）
- **优势**（面试中表现好的地方）
- **待改进**（回答中的薄弱点）
- **改进建议**（针对性练习方向）

历史面试记录可回看，对比进步。

### 3. 简历六维度评分 + AI 优化

上传简历后，AI 从 6 个维度打分：

| 维度 | 满分 |
|------|:--:|
| 教育背景与资质 | 15 |
| 专业技能与技术栈 | 25 |
| 工作经历与项目经验 | 30 |
| 职业发展与稳定性 | 15 |
| 软技能与综合素质 | 10 |
| 简历质量与呈现 | 5 |

**AI 优化**严格遵守"不编造"原则——只优化表达方式，不添加原文不存在的信息，缺数据的地方标注"📝 建议补充"。

### 4. 自动投递 + 防重复

- 按职位关键词 + 城市 + 薪资范围筛选
- HR 活跃度过滤（在线/刚刚活跃/今日活跃）
- 猎头过滤
- 已投递自动跳过（localStorage + SQLite 双重去重）
- 多城市自动轮换
- 随机延迟模拟真人操作

### 5. 投递追踪看板

桌面端仪表盘实时显示：

```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│   127    │ │    23    │ │    5     │ │   12     │
│  总投递  │ │  面试中  │ │  有意向  │ │  已拒绝  │
└──────────┘ └──────────┘ └──────────┘ └──────────┘

📈 近 14 天投递趋势图
📋 投递流水表（公司/岗位/薪资/状态/时间）
🔍 点击查看完整 JD
📥 导出 CSV / JSON / Excel（带格式）
```

### 6. AI 自动回复 HR 消息

实时监听聊天页面，HR 发来消息时：
1. 自动检测对话阶段（打招呼 → 展示 → 问答 → 薪资 → 面试）
2. 结合你的简历和岗位 JD 生成自然回复
3. 三种对话策略可选：激进（主动出击）· 平衡（稳扎稳打）· 保守（谨慎稳妥）

### 7. 安全机制（桌面端 V1.1+）

- **API Key 加密存储**：Fernet 加密，密钥从机器特征派生，`.db` 文件被窃取也无法直接读取
- **本地随机 Token**：首次启动生成 32 位随机 token，外部网页无法调用本地 API
- **CORS 白名单**：仅允许 `localhost` 和 `zhipin.com` 来源
- **频率限制**：AI 端点内置请求频率限制，防止误操作刷爆 Token
- **XSS 防护**：所有 AI 返回内容经 HTML 转义处理
- **隐私清洗**：埋点数据自动过滤敏感字段（密码、API Key、简历文本）

## 🚀 快速开始

### 第一步：安装 Tampermonkey

Chrome / Edge 用户前往扩展商店安装 [Tampermonkey](https://www.tampermonkey.net/)（油猴脚本管理器）。

### 第二步：安装脚本

1. 下载 `Boss_helper.js`
2. 打开 Tampermonkey → 添加新脚本 → 粘贴代码 → `Ctrl+S` 保存
3. 确认脚本已启用，匹配域名 `*://*.zhipin.com/*`

### 第三步：启动桌面端

1. 下载 `boss-desktop-app/Boss海投小助手.exe`
2. 双击运行 → 浏览器自动打开仪表盘
3. 仪表盘 → AI 设置 → 填入 API Key（支持硅基流动/DeepSeek/OpenAI 等）
4. 点击"测试连接"确认可用

### 第四步：开始使用

1. 打开 [BOSS 直聘职位页](https://www.zhipin.com/web/geek/jobs)
2. 脚本自动连接桌面端，日志显示 `🔗 Boss海投小助手已连接`
3. 在脚本面板输入职位关键词（如：前端,开发）、工作地（如：杭州）
4. 点击"启动海投"
5. 切换到桌面端仪表盘查看实时投递流水
